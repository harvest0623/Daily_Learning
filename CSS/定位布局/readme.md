# 文档流
- html中的标签分为三类:
1. 块级元素 (div) : 默认占据一整行
2. 行内块级元素 (input,button): 可以同行展示,也能设置宽高
3. 行内元素 (a,span): 无法被设置宽高

- 浏览器在加载html容器时,默认从上往下,从左往右的排列这些元素

# 定位布局
1. 相对定位: position: relative; 以该容器自身的原本的文档流的位置为参考标准
2. 绝对定位: position: absolute; 以拥有定位属性的父容器为参考标准,如果没有就层层往上找拥有定位属性的父容器,直到找到body标签为止,并以body为参考标准,绝对定位会导致元素脱离文档流
3. 固定定位: position: fixed; 以屏幕的可视区域为参考标准
4. 粘性定位: position: sticky; 以屏幕为参考标准,当页面滚动到阈值时,被定位的元素就像固定在该位置一样
