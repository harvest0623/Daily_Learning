# 弹性布局
1. display: flex; 让一个容器变成弹性容器
2. 弹性容器拥有一个主轴和一个交叉轴，默认主轴为 x 轴
3. flex-direction: column; 可以修改主轴方向
4. justify-content 控制元素在主轴上的排列方式，align-items控制元素在交叉轴上的排列方式
5. flex-wrap 控制子元素在父容器空间不足时可以换行展示
6. align-content 当子元素换行，也就是出现多根轴线时，控制多根轴线之间的排列
7. order 控制子元素排列先后，值默认为 0，越小的值排在越前面
8. flex-grow: 0; 当父容器空间有盈余，该属性可以让子元素默认放大占满父容器
9. flex-shrink: 1; 当父容器空间不足时，该属性可以让子元素默认缩小占满父容器
10. flex-basis: 200px;设置子元素的默认宽度（或者高度）
11. flex: 0 0 200px; 等于上面三个属性的集合
12. align-self: center; 单独修改某一个子元素在交叉轴上的排列方式
# 应用场景
1. 处理居中
2. 三栏布局
